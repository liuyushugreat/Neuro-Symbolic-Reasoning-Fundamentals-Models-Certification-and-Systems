# Contributing to *Neuro-Symbolic Reasoning*

Thank you for your interest in contributing! This is an open academic manuscript; all constructive help is welcome.

## How You Can Contribute

| Type | Where |
|------|-------|
| Typo / grammar fix | Open a **Pull Request** directly |
| Factual error or missing citation | Open an **Issue** with `[errata]` in the title |
| Suggest a new topic or section | Open an **Issue** with `[suggestion]` in the title |
| Add or improve a lab exercise | PR to `experiments/` (MIT-licensed code) |
| Translate a chapter into another language | Open an **Issue** first to coordinate |
| General question or discussion | Use **Discussions** (if enabled) or open an **Issue** |

## Pull Request Guidelines

1. **One PR per topic.** Keep changes focused.
2. **Describe what and why.** A clear PR description helps reviewers.
3. **Respect the dual license.**
   - Book text contributions are released under **CC BY-NC-SA 4.0**.
   - Code contributions are released under **MIT**.
   - By submitting a PR you agree to these terms.
4. **Follow existing formatting.** Markdown style, heading hierarchy, formula delimiters (`$...$` / `$$...$$`), and figure naming conventions should match the rest of the book.
5. **Do not commit large binary files.** Use compressed PNGs for figures; avoid committing PDFs or videos.

## Reporting Errors

If you find a factual, formula, or citation error:

1. Open an Issue titled `[errata] Chapter X — brief description`.
2. Include the chapter file name, line or section, and your proposed correction.
3. If possible, provide a reference (paper, textbook, or authoritative source).

All participants are expected to follow the project's [Code of Conduct](CODE_OF_CONDUCT.md).
